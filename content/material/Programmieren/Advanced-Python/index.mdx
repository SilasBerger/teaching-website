## Parameter und Argumente

## R√ºckgabewerte
Eine Funktion kann zwei Arten von Aufgaben erf√ºllen:
<DefinitionList>
  <dt>1) etwas bewirken</dt>
  <dd>Zum Beispiel bewirkt die Funktion `forward(20)` aus dem Modul `turtle`[^2], dass die Turtle um 20 Pixel nach vorne bewegt wird.</dd>

:::definition[R√ºckgabewert]
Wenn eine Funktion einen Wert produziert, nennen wir diesen Wert den **R√ºckgabewert**.

**Beispiele:**
- Der _R√ºckgabewert_ der `sqrt(n)`-Funktion ist die Quadratwurzel des Parameters `n`.
- Der _R√ºckgabewert_ der `input()`-Funktion ist der gesamte Text, den der Benutzer vor dem Dr√ºcken der Enter-Taste in der Kommandozeile eingibt.
:::

### R√ºckgabewerte zuweisen
Der R√ºckgabewert einer Funktion k√∂nnen wir einer Variable zuweisen:

In diesem Beispielprogramm wird der Benutzerin eine Zahl erfragt und anschliessend deren Quadratwurzel ausgegeben. Dabei werden die R√ºckgabewerte der Funktionen `input()`, `int()`, `sqrt()` und `str()` jeweils einer Variable zugewiesen.

### Einen Wert zur√ºckgeben
:::definition[return]
Eine Funktion hat genau dann einen R√ºckgabewert, wenn ihre letzte Zeile aus einem `return`-Ausdruck besteht.
:::

Mit dem Schl√ºsselwort `return` (dt: _zur√ºckgeben_) wird am Ende einer Funktion ein Wert zur√ºckgegeben. Die Ausf√ºhrung der Funktion endet in jedem Fall, sobald dabei dieser `return`-Ausdruck erreicht wird.

**Beispielprogramm:**
```python showLineNumbers {3-4,7}

    print("Berechne Summe...")
    resultat = a + b
    return resultat
    print("Berechnung erfolgreich!")

print(berechnung)
```

Die Ausgabe dieses Programms sieht wie folgt aus:
```

15
```

- Die Funktion `summe()` berechnet auf der Zeile `3` die Summe aus den Parametern `a` und `b` und weist den _R√ºckgabewert_ dieser Addition der Variable `resultat` zu.
- Der Wert der Variable `resultat` wird mit dem `return`-Ausdruck auf Zeile `4` als _R√ºckgabewert_ der `summe()`-Funktion zur√ºckgegeben.
- Mit dem `return`-Ausdruck auf Zeile `4` endet die Ausf√ºhrung dieser `summe()`-Funktion ‚Äî die Zeile `print("Berechnung erfolgreich!")` wird also nie ausgef√ºhrt und kann daher entfernt werden.
- Auf Zeile `7` wird die `summe()`-Funktion mit den Argumenten `5` und `10` aufgerufen. Der _R√ºckgabewert_ dieses Funktionsaufrufs wird die Zahl `15` sein. Dieser Wert wird der Variable `berechnung` zugewiesen.
- Auf Zeile `8` wird der Wert der Variable `berechnung` (also `15`) ausgegeben.
