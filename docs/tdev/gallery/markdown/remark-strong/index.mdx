# Strong

Dieses Plugin __umrandet__ Textteile. Markdown hat eine gewisse redundanz - fetter Text kann etwa mit den Markierungen `**` oder `__` erzeugt werden.

Dieses Plugin fügt allen `__`-Markierungen einen konfigurierten `className` hinzu. So kann mit etwas CSS etwa ein Rahmen um den Text gezogen werden.

```md
Dieser Text ist __umrandet__.
```

<BrowserWindow>
Dieser Text ist __umrandet__.
</BrowserWindow>

## Konfiguration

`className`
: Der Name des CSS-Klassen, die hinzugefügt werden soll.
: Standard: -
`tagName`
: Der Name des HTML-Tags, das um den Text gezogen werden soll.
: Standard: `strong`

## Installation

:::info[Code]
- `src/plugins/remark-strong`
:::

:::info[`docusaurus.config.ts`]
```ts
import strongPlugin from './src/plugins/remark-strong/plugin';

const REMARK_PLUGINS = [
    /* ... */
    [ strongPlugin, { className: 'boxed' } ]
];
```
:::

:::info[CSS]
CSS für eine __dunkelblaue Umrandung__:

```css
strong.boxed {
    color: #3348b5;
    font-weight: normal;
    border: 1px solid var(--ifm-color-gray-600);
    padding: 1px 0.4em 1px 0.3em;
    border-radius: 0.2em;
    line-height: initial;
}

html[data-theme='dark'] {
    strong.boxed {
        color: #8395ef;
    }
}
```
:::
