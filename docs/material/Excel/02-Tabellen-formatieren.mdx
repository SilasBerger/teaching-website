# Tabellen formatieren
In diesem Abschnitt lernen Sie, wie Sie eine Tabelle in Excel formatieren können.

::::aufgabe[Formatieren]
<TaskState id="d0c1f2b6-3a4e-4b5c-8d7f-9e0a1c5f3b2c" />
Laden Sie die Excel-Datei _«Statistik der Jugendstrafrechtspflege des Kantons Zürich.xslx»_ herunter (siehe OneDrive-Ordner), öffnen Sie sie und gehen Sie auf das Blatt **Sanktionen**. Formatieren Sie das Blatt anschliessend wie folgt:
- Die Zellen der ersten Zeile sollen als Überschriften erkennbar sind. Verwenden Sie dazu die Möglichkeiten im Menü __Schriftart__ (__fett__, __Rahmenlinien__, etc.).
- Fügen Sie oberhalb der Tabelle ein neue Zeile ein (__:mdi[cursor-default-click] Rechtsklick__ auf Zahl __1__ zuvorderst bei erster Zeile → __Zellen einfügen__).
- Schreiben Sie auf der neuen Zeile __1__ einen aussagekräftigen Titel und formatieren Sie diesen, so dass er als Titel erkennbar ist.

:::tip[Mehrere Zellen gleichzeitig formatieren]
Sie können mehrere Zellen gleichzeitig formatieren, wenn Sie diese zuerst markieren.
:::
