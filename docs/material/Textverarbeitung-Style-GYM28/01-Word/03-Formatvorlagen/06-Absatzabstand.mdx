# Absatzabstand anpassen

Um den Absatzabstand auf einen bestimmten Wert zu setzten, gehen Sie folgendermassen vor:

::::Tabs :::Tab[Windows]{value="win"} Öffnen Sie das [Änderungs-Menu der entsprechenden Formatvorlage](Formatvorlagen-Anpassen#option-2-vorlagen-ändern) der entsprechenden Formatvorlage.

Klicken Sie unten rechts auf **Format** → **Absatz**: ![--width=500px](img/win/format-absatz.png)

Hier können Sie im entsprechenden Feld den **Abstand vor** oder den "Abstand nach" anpassen. Anschliessend bestätigen Sie mit **OK**. ![--width=500px](img/win/abstand.png) :::

:::Tab[macOS]{value="macos"} Öffnen Sie das [Änderungs-Menu der entsprechenden Formatvorlage](Formatvorlagen-Anpassen#option-2-vorlagen-ändern) der entsprechenden Formatvorlage.

Klicken SIe unten rechts auf **Format** → **Absatz**: ![--width=450px](img/macos/absatz.png)

Hier können Sie im entsprechenden Feld den **Abstand vor** oder den **Abstand nach** anpassen. Anschliessend bestätigen Sie mit **OK**. ![--width=450px](img/macos/abstand.png) ::: ::::
