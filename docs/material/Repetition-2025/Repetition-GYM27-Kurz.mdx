# Rückblick Informatik
Ihr Informatikunterricht ist vorbei. Zeit für einen Rückblick!

Dazu erstellen Sie als Gruppe einige Kahoot-Fragen. Herr Berger wird die besten Fragen auswählen und daraus ein Kahoot-Quiz erstellen, das wir anschliessend spielen können.

## Auftrag
<Steps>
1. Dreiergruppen bilden
2. Ein Person pro Gruppe: [Kahoot Excel-Vorlage](https://erzbe-my.sharepoint.com/:x:/g/personal/silas_berger_gbsl_ch/EcCANadtYt9GtcRs7WyAZEEBLlZyZur08ADnuRRSs_KYbA?e=VihQoq) **herunterladen** (__Datei__ → __Eine Kopie erstellen__ → __Eine Kopie herunterladen__)
3. Die heruntergeladene Excel-Datei so abspeichern, dass Sie sie später wiederfinden können (z.B. in Ihrem OneDrive).
4. Mindestens 5 gute Kahoot-Fragen zu einem beliebigen Thema erstellen, das wir im Informatikunterricht behandelt haben (Hinweise unten beachten!)
5. **Bis Ende Lektion:** Die fertig ausgefüllte Excel-Datei speichern, schliessen und per Teams an Herrn Berger senden.
</Steps>

### Hinweise zur Kahoot Excel-Tabelle
- Lassen Sie die Spaltenüberschriften, etc. unverändert.
- Beachten Sie die maximale Länge der Zeichen für Fragen und Antworten.
- Geben Sie für jede Frage 2-4 Antwortmöglichkeiten in die entsprechenden Spalten ein.
- Geben Sie in der letzten Spalte die Nummer(n) der 1-4 richtigen Antwort(en) an (z.B. `2` wenn Antwort 2 richtig ist, `1,3` wenn Antworten 1 und 3 richtig sind, etc.).
