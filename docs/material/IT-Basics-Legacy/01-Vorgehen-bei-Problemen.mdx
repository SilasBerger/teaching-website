# Vorgehen bei Problemen

:::Hero ![](./img/erste-hilfe.jpeg) ::: :::Caption Bild von <a href="https://unsplash.com/photos/orange-erste-hilfe-med-kit-j-ped4HD32Q?utm_content=creditCopyText&utm_medium=referral&utm_source=unsplash">Ploegerson (Unsplash)</a> :::

Bei der Arbeit am Computer treten oft Probleme auf - das gehört zum ganz normalen Alltag. Damit Sie in solchen Momenten nicht in Ihrer Arbeit blockiert sind, ist es wichtig, dass Sie sich mit der Zeit gewisse Strategien zur Selbsthilfe aneignen.

Hier erfahren Sie, wie Sie bei Problemen am besten vorgehen.

## Problem analysieren

Sollte bei Ihnen also mal etwas nicht funktionieren, dann müssen Sie Ihr Problem als Erstes sorgfältig analysieren. Dazu sollten Sie versuchen, folgende Fragen so präzise wie möglich zu beantworten:

1. Was möchten Sie erreichen? Was ist das erwartete Ergebnis?
2. Was passiert (nicht)? Was ist das tatsächliche Ergebnis?
3. Wie unterscheiden sich diese beiden Ergebnisse? Sprich: was genau funktioniert nicht? Inwiefern funktioniert es nicht?

Auf diese Weise erhalten Sie eine konkrete, genaue Problembeschreibung. Verwenden Sie diese für alle Lösungsschritte im nächsten Abschnitt.

## Lösung suchen

Sie haben jetzt eine konkrete Problembeschreibung. Damit können Sie nun die folgenden Lösungsschritte der Reihe nach durcharbeiten:

1. Suchen Sie in Ihren Unterlagen und Referenzen nach einer Ressource, die Ihnen bei Ihrem Problem helfen könnte.
2. Recherchieren Sie das Problem mit Google (vgl. [Richtig Googeln](Suchmaschinen#richtig-googlen))
3. Suchen Sie sich Hilfe, zum Beispiel bei Kolleg\*innen oder bei Ihrer Lehrperson. Wenn Sie ein Problem nicht selbstständig Lösen können, dann dürfen Sie sich immer Hilfe holen. Dabei ist es aber wichtig, dass Sie das Problem möglichst genau beschreiben können. Die Problemanalyse aus dem ersten Abschnitt wird Ihnen dabei behilflich sein.
