# Einen Plan erstellen
## Excalidraw
Sie sind sowohl mit Excalidraw als auch mit Flowcharts bereits vertraut. Für dieses Projekt sollten Sie Excalidraw allerdings nicht in Classrooms, sondern via https://excalidraw.com/ verwenden.

Im folgenden Video erhalten Sie einen Refresher zu Flowcharts und Exclaidraw, und Sie lernen, wie Sie Ihre Flowcharts exportieren können, um sie mit ihrem Gruppenmitglieder zu teilen und im Abgabeordner zu speichern.

::youtube[https://www.youtube.com/embed/MPwM1PcKONU?si=V5lAESAHgIUGp-kU]

## Das Spiel planen
In der inhaltlichen Gestaltung Ihres Text Adventures sind Sie völlig frei. Überlegen Sie sich also, welche Geschichte Sie erzählen möchten, und welche Entscheidungen die Spieler:innen treffen können sollen.

Erfahrungsgemäss eigenen sich folgende Arten von Geschichten besonders gut für Text Adventures:
- Suche (nach einem Schatz, einer Person, einer Antwort, dem Sinn des Lebens…)
- Flucht (aus einem Gefängnis, aus einer gefährlichen Situation, …)
- Detektivgeschichte / Krimi
- Abenteuerreise / Expedition

### Auftrag
Überlegen Sie sich eine passende Geschichte und erstellen Sie dazu einen ersten Entwurf eines Flowcharts in Excalidraw. Darauf sollen alle Entscheidungspunkte und möglichen Enden Ihres Spiels ersichtlich sein. Ihr Flowchart darf sich natürlich im Verlauf des Projekts noch ändern, muss am Schluss aber den finalen Spielverlauf korrekt abbilden.

:::tip[Anforderungen an das Spiel]
Damit Sie wissen, worauf Sie bei der Planung Ihres Spiels achten müssen, finden Sie [hier](./03-Anforderungen-Spiel.mdx) die Anforderungen an das Spiel.
:::

Denken Sie dran, die Excalidraw-Datei so zu speichern, dass alle Gruppenmitglieder darauf zugreifen können.
