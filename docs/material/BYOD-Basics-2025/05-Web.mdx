# Modul 3: Web
:::info[Lernziele]
1. Sie können erklären, was ein Browser ist und wozu man ihn braucht.
2. Sie können mindestens drei verschiedene Browser aufzählen und anhand mindestens zweier Kriterien miteinander vergleichen.
3. Sie können erklären, weshalb die Browser-Version von OneDrive nicht das gleiche ist wie die OneDrive-Integration in Ihrem Dateisystem.
4. Sie können den Unterschied zwischen «Google» als Suchmaschine und «Google Chrome» als Browser erklären.
5. Sie können eine Webseite öffnen, ohne dabei eine Suchmaschine zu verwenden.
6. Sie können eine Datei von einer Webseite herunterladen und diese auf Ihrem Gerät wiederfinden.
7. Sie können ein Programm von einer Webseite herunterladen, installieren, starten und wieder deinstallieren.
:::
