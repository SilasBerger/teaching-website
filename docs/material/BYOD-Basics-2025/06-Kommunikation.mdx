# Modul 4: Kommunikation
:::info[Lernziele]
1. Sie können ein E-Mail an eine:n bestimmte:n Empfänger:in senden und dabei alle Aspekte der «E-Mail-Netiquette» korrekt umsetzen.
2. Sie können bei einem E-Mail weitere Empfänger:innen im Cc und im Bcc anfügen und können erklären, was es mit diesen Feldern auf sich hat.
3. Sie können eine Teams-Nachricht an eine:n bestimmte:n Empfänger:in senden und dabei alle Aspekte der «Teams-Netiquette» korrekt umsetzen.
4. Sie können eine Datei per Teams an eine:n bestimmte:n Empfänger:in senden und achten vor dem Versenden auf eine sinnvolle Benennung der Datei. **Achtung:** Bei der Probe werden Sie Ihr Video auch so abgeben müssen!
5. Sie können in Teams einen Audio- oder Videoanruf starten und darin Ihren Bildschirm teilen.
:::

## E-Mail
- Netiquette: Betreff, Anrede, Grussformel; != WhatsApp
- Empfänger, Cc, Bcc
