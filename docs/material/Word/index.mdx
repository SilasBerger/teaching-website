# Textverarbeitung mit Microsoft Word
Textverarbeitung ist die _Erstellung und Bearbeitung von schriftlichen Texten mithilfe von organisatorischen und technischen Mitteln_[^1]. Textverarbeitungsprogramme findet man in fast jedem sogenannten _Office-Paket_ wie zum Beispiel _Microsoft Office_ oder _Google Workspace_[^2]. In diesem Kapitel befassen wir uns mit dem vermutlich wichtigsten und bekanntesten Textverarbeitungsprogramm, nämlich **Microsoft Word**.

:::insight[Geschichte von Microsoft Word]
Die erste Version von Microsoft Word wurde bereits im November 1983 veröffentlicht. Damals war es nur für Computer mit dem _MS-DOS_-Betriebssystem (einem Vorläufer von Microsoft Windows) verfügbar. Die erste Version für die _Macintosh_-Computer von Apple folgte im Jahr 1985.
:::

[^1]: Quelle: [Wikipedia](https://de.wikipedia.org/wiki/Textverarbeitung)
[^2]: Das Textverarbeitungsprogramm im _Google Workspace_ Office-Paket kennen Sie vermutlich unter dem Namen _Google Docs_.
