# Formatvorlagen
Formatvorlagen sind ein mächtiges Werkzeug in Microsoft Word, das Ihnen dabei hilft, Ihre Dokumente professionell und einheitlich zu gestalten. Sie ermöglichen es Ihnen, die Gestaltung Ihrer Dokumente zu vereinheitlichen und gleichzeitig den Aufwand für die Formatierung zu minimieren.



:::key[Warum Formatvorlagen]
Die Verwendung von Formatvorlagen ist aus mehreren Gründen unverzichtbar.

* Sie sparen viel Zeit, da Sie nicht jede Überschrift oder jeden Absatz manuell formatieren müssen; stattdessen können Sie mit einem Klick eine konsistente Gestaltung anwenden.
* Formatvorlagen ermöglichen ein durchgängig professionelles Erscheinungsbild Ihres Dokuments. Wenn Sie das Design einer Formatvorlage ändern, wird diese Änderung automatisch im gesamten Dokument übernommen.
* Formatvorlagen helfen Ihnen, schnell durch mehrseitige Dokumente zu navigieren.

Die **direkte Formatierung** (Anpassungen der Schriftart, Schriftgrösse, Schriftfarbe, etc. ohne die Verwendung von Formatvorlagen) sollten Sie deshalb **unbedingt unterlassen**.
:::
