# ⭐️ Binäre Addition
Wie funktioniert eigentlich Addition im Binärsystem?

::::aufgabe[Binäre Addition ausprobieren]
<TaskState id="97234333-aea6-4c65-bfa0-c1904d9d10ad" />
Schauen Sie sich [diese Seite](https://oinf.ch/interactive/addition-in-basis-2/) an. Sie könen auf die roten Zahlen klicken, um zwischen $0$ und $1$ umzuschalten.

Überlegen Sie sich, wie die Addition im Binärsystem funktionieren könnte. Halten Sie Ihre Lösung hier fest.

<QuillV2 id="d6aae58e-961e-462a-b52b-a2200833a462" />
::::

:::aufgabe[Binäre Addition üben]
<TaskState id="58b0dbed-db44-4b24-bca0-262c098764cc" />
