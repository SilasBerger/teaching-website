# ::circuitverse

Bindet einen [Circuitverse](https://circuitverse.org/) Schaltkreis ein.

```mdx
::circuitverse[https://circuitverse.org/simulator/embed/rothe-inverter]
```

<BrowserWindow>::circuitverse[https://circuitverse.org/simulator/embed/rothe-inverter]</BrowserWindow>

## Optionen

`height` : Höhe des iframes. : Standardmässig auf **315px** gesetzt. : z.B. `height=200px` `width` : Breite des iframes. : Standardmässig auf **100%** gesetzt. css Eigenschaften : Es können beliebige css-Eigenschaften in Form `eigenschaft=wert` angegeben werden, die auf das iframe angewendet werden.

```mdx
::circuitverse[https://circui...]{width=300px height=200px border="2px solid red"}
```

<BrowserWindow>
::circuitverse[https://circuitverse.org/simulator/embed/rothe-inverter]{width=300px height=200px border="2px solid red"}
</BrowserWindow>
