## Fairplay
- Die geplante Verwendung von KI (ChatGPT, Claude, Microsoft / VSCode Copilot, etc.) muss explizit und präzise im Projektvertrag festgehalten werden. Die Verwendung von KI ist ausschliesslich im dadurch festgelegten Rahmen erlaubt und muss in der Projektdokumentation transparent gemacht werden. 
- Die Verwendung externer Hilfe (z.B. Eltern, Geschwister, Freund:innen) folgt denselben Regeln wie die Verwendung von KI.
- Besteht der Verdacht, dass die Gruppe gegen eine oder mehrere der Fairplay-Regeln verstossen hat, so wird die Gruppe zu einem Gespräch aufgeboten, in dem die Lehrperson die Kenntnisse der Gruppenmitglieder überprüft, um eine differenzierte Beurteilung vornehmen zu können.
