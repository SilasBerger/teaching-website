# Das div-Element
Für einige Styling-Zwecke müssen wir unsere Webseite in verschiedene Bereiche unterteilen. Das ist beispielsweise nötig, wenn wir:
- bestimmte Bereiche mit einem Hintergrund, einem Rahmen, einem Abstand, einem Schatten, etc. versehen wollen.
- mehrere «Inhalts-Behälter» nebeneinander anordnen wollen.
- bestimmte Bereiche mit einem bestimmten Layout versehen wollen, z.B. mit einem mehrspaltigen Layout.
