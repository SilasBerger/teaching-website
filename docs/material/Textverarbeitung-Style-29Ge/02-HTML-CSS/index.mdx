# HTML & CSS
Sie haben gelernt, wie Sie mit Microsoft Word Texte erstellen, strukturieren und formatieren können. Das Ergebnis daraus ist ein digitales oder physisches Dokument, also ein PDF oder ein ausgedrucktes Papier, in einem festgelegten Format (z.B. A4 Hochformat).

Nebst physischen und digitalen Dokumenten gibt es aber noch weitere Bereiche, in denen wir Texte erstellen, strukturieren und formatieren möchten. Ein ganz zentraler solcher Bereich ist das Web: eine (einfache) Webseite ist nämlich nichts anderes als ein strukturierter, formatierter Text!

In diesem Kapitel lernen Sie, wie Sie die formalen Sprachen _HTML_ und _CSS_ nutzen können, um einen Text für das Web zu erstellen und zu formatieren.

Dieser Auftrag ist eine **Einzelarbeit**.

## VSCode
Visual Studio Code (_VSCode_) kennen Sie bereits aus Ihrer Arbeit mit Python. Es handelt sich dabei aber nicht lediglich um eine Entwicklungsumgebung für Python, sondern um einen sehr vielseitigen Code-Editor, der grundsätzlich für praktisch alle Sprachen geeignet ist. Diesen können Sie dementsprechend auch für die Sprachen des Web (HTML, CSS) verwenden.

Im folgenden Tutorial lernen Sie, wie Sie VSCode für die HTML-Entwicklung verwenden. Am besten probieren Sie es gleich mit aus. Den Code dazu finden Sie direkt unten dran.

::youtube[https://www.youtube.com/embed/p7cTMWYiZNE?si=suxS8MHwxa-Kw-6U]

```html
<!DOCTYPE html>
<html>
  <body>
    <h1>VSCode Demo</h1>
    <p>Hallo! Ich entwickle eine HTML-Seite in VSCode!</p>
  </body>
</html>
```
