# Ein HTML-Dokument erstellen
Wir Sie im Video in der Einleitung bereits gelernt haben, nutzen wir VSCode, um unseren HTML- und CSS-Code zu schreiben. Falls Sie das bisher noch nicht gemacht haben, dann probieren Sie es nun hier aus.

Hier dazu nochmal das Video:

::youtube[https://www.youtube.com/embed/p7cTMWYiZNE?si=suxS8MHwxa-Kw-6U]

:::aufgabe[Ein HTML-Dokument erstellen]
<ProgressState id="afa9bc90-d1b0-454a-90fb-3b8326f4a685" confirm keepPreviousStepsOpen>
  1. Erstellen Sie in Ihrer OneDrive einen neuen Ordner, z.B. `VSCode HTML Demo`. Öffnen Sie diesen anschliessend in VSCode.
  2. Erstellen Sie in diesem Ordner eine neue Datei mit dem Namen `index.html` und schreiben Sie in diese Datei den folgenden HTML-Code hinein:
     ```html
     <!DOCTYPE html>
     <html>
       <body>
         Hallo, Welt!
       </body>
     </html>
     ```
  3. Speichern Sie die Datei. Öffnen Sie die Datei `index.html` danach in einem Webbrowser, entweder via Finder / Explorer, oder mit der _Live Server_ Extension in VSCode. Kontrollieren Si, ob die Webseite so angezeigt wird:
     <BrowserWindow url="C:\Benutzer\silas\Webseite\index.html">
      ![](img/hallo-welt-unformatiert.png)
     </BrowserWindow>
  4. Ändern Sie etwas am HTML-Code ab (z.B. `Hallo, Welt!` zu `Hallo, VSCode!`), speichern Sie die Datei und aktualisieren Sie die Seite im Browser, um die Änderung zu sehen.
</ProgressState>
:::
