# Zwischenabgabe
Zum Ende dieses Unterrichtsblocks **(17:30 Uhr)** reichen Sie den ersten Entwurf des Flow Charts für Ihre Story ein.

:::info[Entwurf]
Diese Zwischenabgabe ist nur ein Entwurf. Sie dürfen Ihr Flow Chart natürlich noch weiter entwickeln und anpassen.

Speichern Sie die neuen Versionen aber jeweils unter einem neuen Namen ab (z.B. `...v1.excalidraw`, `...v2.excalidraw`, etc.). Es sollte immer möglich sein, diesen Entwurf nochmal anzuschauen.

## Anleitung zur Abgabe
Für die Zwischenabgabe erledigen Sie die folgenden Schritte **einmal pro Gruppe** (nicht pro Person).

1. Erstellen und **kopieren** Sie einen OneDrive-Freigabelink für Ihren Gruppenordner.
   - Wählen Sie die Sichtbarkeit so, dass **brr und brs** darauf zugreifen können.
   - **Achtung:** Klicken Sie **nicht** auf _Senden_, sondern auf _Link kopieren_!
   - Eine Anleitung dazu finden Sie [hier (für OneDrive Online)](https://support.microsoft.com/de-de/office/freigeben-von-onedrive-dateien-und-ordnern-9fcc2f7d-de0c-4cec-93b0-a82024800c07).
2. Füllen Sie [diese Forms-Umfrage aus](https://forms.office.com/Pages/ResponsePage.aspx?id=Y4MGSWGDB0aVSWK2tVeUqneR4FxJ7SlIvbWmmL8iOgxUMTVEMDlQVTBBTjFHSUpYT004SjlKOTdFRC4u). 
   - Hier fügen Sie unter anderem den zuvor kopierten OneDrive-Freigabeklink ein.
