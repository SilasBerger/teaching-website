# Aufgaben: SELECT 1
:::warning[Arbeitsanweisungen]
- Erstellen Sie zu jeder Aufgabe eine passende SQL-Abfrage (`SELECT ...`), welche das beschriebene Ergebnis erzeugt.
- Öffnen Sie für jede Abfrage jeweils einen **neuen SQL-Editor** (**überschreiben Sie nicht Ihre vorherige Lösung**)
- **Speichern** Sie jede Abfrage als `SELECT 1 Abfrage [Nummer].sql` (also z.B. `SELECT 1 Abfrage 1.sql`, dann `SELECT 1 Abfrage 2.sql`, und so weiter) auf Ihrer OneDrive, damit Sie sie später wieder finden.
:::

:::aufgabe[SQL-Abfragen erstellen]
_**Hinweis**: Lesen Sie die obigen Arbeitsanweisungen genau durch und erstellen Sie für jede dieser 8 Abfragen eine neue `.sql`-Datei!_

Erstellen Sie SQL-Abfragen mit folgenden gewünschten Resultaten:
1. Alle Spalten aller Haustiere (Tabelle `haustiere`).
2. Alle Spalten des Haustiers mit dem Namen _Fluffy_.
3. Nur die Spalten `name`, `tierart` und `lieblingsfutter` von _Fluffy_.
4. Die Spalten `name`, `tierart` und `lieblingsfutter` aller Haustiere der Tierart Hund.
5. Die Spalten `name`, `tierart` und `lieblingsfutter` aller Haustiere, deren Lieblingsfutter Körner sind.
6. Die Spalten `name`, `tierart` und `lieblingsfutter` aller Hunde, deren Lieblingsfutter Leckerli sind.
7. Die **Anzahl** aller Haustiere, die entweder ein _Hund_ sind, oder deren Lieblingsfutter _Körner_ sind.
8. Die Anzahl aller Haustiere, welche beide dieser Bedingungen erfüllen:
   1. Ihr Lieblingsfutter sind **nicht** Leckerli.
   2. Es sind entweder Hunde oder sie haben Körner als Lieblingsfutter.
:::
