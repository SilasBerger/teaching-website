# SQLiteStudio Installieren

**SQLiteStudio** ist ein kostenloses Programm, mit welchem wir SQLite-Datenbanken verwalten können.

Wählen Sie Ihr Betriebssystem (Windows oder macOS) aus und befolgen Sie die Installationsanleitung.

::::Tabs{groupId="os"} :::Tab[Windows]{value="win"}

1. Öffnen Sie die Download-Webseite: [https://github.com/pawelsalawa/sqlitestudio/releases](https://github.com/pawelsalawa/sqlitestudio/releases).
2. Klicken Sie auf den folgenden blauen Download-Link, um SQLiteStudio für Windows herunterzuladen: ![--width=500px](img/install/win/01-download.png)
3. Führen Sie die heruntergeladene Datei aus, indem Sie sie im Downloads-Ordner suchen und darauf **doppel**klicken: ![--width=400px](img/install/win/02-open.png)
4. Klicken Sie bei der folgenden Sicherheitswarnung auf "Ja": ![--width=400px](img/install/win/03-allow.png)
5. Folgen Sie dem Installationsprogramm, indem Sie jeweils auf "Next" klicken. Akzeptieren Sie die Lizenz und belassen Sie alle anderen Einstellungen unverändert. ![--width=400px](img/install/win/04-accept.png)
6. Wenn Sie SQLiteStudio zum ersten Mal öffnen, wählen Sie die Sprache _Deutsch_ aus: ![--width=400px](img/install/sprache.png) :::

:::Tab[macOS]{value="macos"}

1. Öffnen Sie die Download-Webseite: [https://github.com/pawelsalawa/sqlitestudio/releases](https://github.com/pawelsalawa/sqlitestudio/releases).
2. Klicken Sie auf den folgenden blauen Download-Link, um SQLiteStudio für Windows herunterzuladen: ![--width=500px](img/install/macos/01-download.png)
3. Führen Sie die heruntergeladene Datei aus, indem Sie sie im Downloads-Ordner suchen und darauf **doppel**klicken. Es öffnet sich folgendes Fenster, in welchem Sie das Box-Symbol ebenfalls nochmal **doppel**klicken müssen: ![--width=400px](img/install/macos/02-open.png)
4. Sie erhalten dabei vermutlich eine Fehlermeldung, weil das Programm nicht von einem verifizierten Entwickler stammt. Öffnen Sie in dem Fall Ihre **Systemeinstellungen**, gehen Sie zu **Datenschutz & Sicherheit**, suchen Sie den Eintrag wie im Bild gezeigt, und klicken Sie dort auf "Dennoch öffnen": ![--width=400px](img/install/macos/03-open-anyway.png)
5. Klicken Sie bei diesem Dialog dann nochmal auf "Öffnen". ![--width=400px](img/install/macos/04-really-open.png)
6. Folgen Sie dem Installationsprogramm, indem Sie jeweils auf "Next" klicken. Akzeptieren Sie die Lizenz und belassen Sie alle anderen Einstellungen unverändert. ![--width=400px](img/install/macos/05-accept.png)
7. Wenn Sie SQLiteStudio zum ersten Mal öffnen, wählen Sie die Sprache _Deutsch_ aus: ![--width=400px](img/install/sprache.png) ::: ::::
