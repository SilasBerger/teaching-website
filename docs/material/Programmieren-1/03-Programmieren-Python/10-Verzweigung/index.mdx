# Verzweigung
Sie sind bereits vertraut mit der `while`-Schleife: Sie wiederholt ihren KÃ¶rper so lange, bis die gegebene Bedingung nicht mehr zutrifft. Hier nochmal ein Beispiel zur Wiederholung:

```py live_py
from random import *

zufallszahl = randint(0, 10)
counter = 0
while counter < zufallszahl:
  print('Counter ist kleiner als ' + str(zufallszahl))
  counter = counter + 1
print('Counter ist nicht mehr kleiner als ' + str(zufallszahl))
```
